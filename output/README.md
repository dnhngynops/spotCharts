# Output Directory

This directory contains generated reports and data exports.

## Contents

- **HTML Tables**: Formatted playlist data tables
- **CSV Files**: Raw data exports
- **PDF Reports**: Executive summary reports (future)
- **Excel Workbooks**: Multi-tab analytics workbooks (future)

## File Naming Convention

Files are timestamped for easy identification:

```
spotify_charts_YYYYMMDD_HHMMSS.html
spotify_charts_YYYYMMDD_HHMMSS.csv
weekly_report_YYYYMMDD.pdf
```

## Notes

- This directory is gitignored
- Generated files are also uploaded to Google Drive
- Old files can be safely deleted after upload
